Linux将python默认python2修改为python3

2019-12-24教程

kali Linux将python修改为默认python3

现在python2接近2020年的停止维护时间了，可以说python2已经被淘汰掉了，而不少Linux系统上还默认是python2，而我们使用时每次都要刻意输入python3，感觉有点繁琐，作为一个懒癌患者，自然是无法容忍的。

查看默认python是什么版本：

输入：python -V

查看python路劲：

which python
显示是在 /usr/bin/python

查看所有的python文件

ls -al /usr/bin/python*

删除当前python文件

sudo rm -rf /usr/bin/python

替换为python3

ln -s /usr/bin/python3 /usr/bin/python

再次查看默认python是什么版本：

输入：python -V

到此已完成了。
