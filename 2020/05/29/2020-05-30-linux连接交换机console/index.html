linux连接交换机console

2020-05-29教程

自从用了linux做主力系统之后，有很多习惯都要改变了，今天想连接一台交换机，结果方法跟windows一点也不相同，我也是通过网络上的教程才连上的，特意写个自己总结后的文章，方便自己后面查看。

第一步：查看usb端口名称

sudo dmesg | grep usb

第二步：安装minicom

sudo apt install minicom
sudo yum install minicom
sudo pacman -S minicom

manjaro 可以通过软件商店安装

第三步：打开minicom设置

sudo minicom -s

        +-----[configuration]------+
        | Filenames and paths      |
        | File transfer protocols  |
        | Serial port setup        |
        | Modem and dialing        |
        | Screen and keyboard      |
        | Save setup as dfl        |
        | Save setup as..          |
        | Exit                     |
        | Exit from Minicom        |
        +--------------------------+

选中 | Serial port setup | 回车

+-----------------------------------------------------------------------+
| A -    Serial Device      : /dev/ttyUSB0                              |
| B - Lockfile Location     : /var/run                                  |
| C -   Callin Program      :                                           |
| D -  Callout Program      :                                           |
| E -    Bps/Par/Bits       : 9600 8N1                                  |
| F - Hardware Flow Control : No                                        |
| G - Software Flow Control : No                                        |
|                                                                       |
|    Change which setting?                                              |
+-----------------------------------------------------------------------+

选择A 修改相应的usb名称
选择E 修改波特率9600
选择F 将Yes改为No

我的设置如下：
A - Serial Device : /dev/ttyUSB0 我的机器只有一个串口，所以为0
B - Lockfile Location : /var/run lockfile路径默认
C - Callin Program :
D - Callout Program :
E - Bps/Par/Bits : 9600 8N1 设置波特率 9600
F - Hardware Flow Control : No 关闭硬件流量控制
G - Software Flow Control : No 关闭软件流量控制

设置完后按回车确认。

第四步：保存设置

选中 | Save setup as dfl | 回车
选中 | Exit from Minicom | 回车退出

第五步：连接交换机console

sudo minicom

注意：连接之后可能没有反应，按几下回车便有显示。

连接界面如下：
