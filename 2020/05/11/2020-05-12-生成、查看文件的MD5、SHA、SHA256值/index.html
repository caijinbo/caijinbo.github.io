linux 生成、查看文件的MD5、SHA、SHA256值

2020-05-11教程

Linux系统生成MD5、SHA、SHA256

md5sum filename >> MD5.txt

sha1sum filename >> SHA1.txt

sha256sum filename >> SHA256.txt

Linux系统查看 MD5、SHA、SHA256

md5sum filename

sha1sum filename

sha256sum filename
